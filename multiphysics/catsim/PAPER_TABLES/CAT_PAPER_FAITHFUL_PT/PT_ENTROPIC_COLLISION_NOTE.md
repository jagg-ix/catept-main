## PT-symmetric bookkeeping export

From the fitted paper-style CAT/EPT visibility law

- **Visibility model**:  \(V(S)=V_{\mathrm{cl}}(\eta)\,\exp(-\lambda_{\mathrm{ent}}|S|)\)
- **Classical visibility factor**:  \(V_{\mathrm{cl}}(\eta)=\frac{2\eta}{1+\eta^2}\)

we define a derived quantity

- \(q(S)\equiv -\ln\big(V(S)/V_{\mathrm{cl}}\big)=\lambda_{\mathrm{ent}}|S|\)

This q(S) is used to export a **toy 2x2 PT-symmetric representation**:

- \(Q = q\,\sigma_y\)
- \(\eta = e^{-Q}\)  (metric operator)
- \(P = \begin{pmatrix}0&1\\1&0\end{pmatrix}\)
- \(C = e^{Q} P\)

The corresponding matrix entries are written to `pt_metric_tables.csv`.

## Relation to entropic proper time and step control

In the CAT/EPT implementation used in this repo, the entropic proper time increment is

- \(d\tau_{\mathrm{ent}} = \lambda\,dt\)

Numerically, the code can enforce a CFL/Nyquist-style ceiling by tying a reference rate
\(\lambda_0\) to the simulation timestep \(\Delta t\), so that \(\lambda_0\sim \mathrm{safety}/\Delta t\).

## Relation to a linear Boltzmann collision operator (relaxation form)

A common linearized (relaxation-time) Boltzmann collision model is of the form

- \(\mathcal{L}_{\mathrm{coll}}[f] = -\nu\,(f-f_{\mathrm{eq}})\)

where \(\nu\) is a collision frequency. In Drude-like optics this \(\nu\) is closely related to
\(1/\tau\) (the Drude scattering rate).

In this repo’s visibility-only CAT/EPT layer, \(\lambda_{\mathrm{ent}}\) plays an analogous **rate** role
for coherence loss (it controls decay of the interference cross term vs separation).

Therefore, when you want a “collision-operator” interpretation, treat:

- \(\nu_{\mathrm{eff}} \leftrightarrow \lambda_{\mathrm{ent}}\)

as a *modeling identification*, and verify it empirically by checking whether the same
rate also predicts the extracted rise/decay parameters from the time-domain table.
