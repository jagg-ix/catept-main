## 🚀 QUICK START GUIDE

### 1. Download & Extract

```bash
# Download CATEPT-Complete-v3.1.zip

# Verify integrity
md5sum -c CATEPT-Complete-v3.1.zip.md5
sha256sum -c CATEPT-Complete-v3.1.zip.sha256

# Extract
unzip CATEPT-Complete-v3.1.zip
cd CATEPT-Complete-v3.1
```

### 2. Run Quickstart

```bash
# Interactive setup and system check
./quickstart.sh
```

This checks:
- ✅ Python installation
- ✅ Dependencies
- ✅ Repository structure
- ✅ Optional tools (Wolfram, Lean4)
- ✅ Quick tests

### 3. Choose Your Path

**Path A: Read the Paper**
```bash
open paper/main.pdf  # macOS
xdg-open paper/main.pdf  # Linux
```

**Path B: Run Verifications**
```bash
# All systems
./tools/mathematica/run_all_verifications.sh

# Python only
cd verification/python && python3 verify_all.py --verbose

# Mathematica only
python3 tools/mathematica/verify_mathematica.py --verbose
```

**Path C: Run Simulations**
```bash
cd simulations/catsim

# Quick check
make unit_check

# Full reproduction
make repro_from_xlsx
```

**Path D: Use Docker**
```bash
# Start development environment
docker-compose up catept

# Run verifications in container
docker-compose run python-verify

# Start Jupyter
docker-compose up jupyter
# Then open http://localhost:8888
```

## 🔧 INSTALLATION

### Basic Installation

```bash
# Extract archive
unzip CATEPT-Complete-v3.1.zip
cd CATEPT-Complete-v3.1

# Run quickstart
./quickstart.sh

# Install Python dependencies
pip install -r verification/python/requirements.txt
```

### Docker Installation

```bash
# Build containers
docker-compose build

# Start development environment
docker-compose up catept
```

### Full Installation

```bash
# Python packages
pip install numpy scipy sympy matplotlib jupyter pytest

# Simulation packages
cd simulations/catsim
pip install -r requirements-ui.txt
pip install -e '.[qutip,kwant,oqupy]'

# Optional: Wolfram Engine
# Download from: https://www.wolfram.com/engine/

# Optional: Lean4
curl https://raw.githubusercontent.com/leanprover/elan/master/elan-init.sh -sSf | sh
```
