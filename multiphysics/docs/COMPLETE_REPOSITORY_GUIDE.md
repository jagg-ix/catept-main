## 📖 Usage Examples

### Python

```python
from verification.python.core import registry

# Get and verify equation
eq = registry.get_by_label('eq:complex_action')
print(eq.sympy_expression())  # S_R + I*S_I
print(eq.verify_positivity())  # True

# Export to other languages
print(eq.mathematica_code())
print(eq.lean_statement())
```

### Mathematica

```mathematica
(* Load package *)
Get["verification/mathematica/core/CAT_EPT_Unified_v1.0.wl"]

(* Complex action and Hamiltonian *)
S = SR + I*SI;
H = HR - I*HI;

(* Entropic time *)
τent = Integrate[λ[t], t];

(* GKLS master equation *)
dρdt = -I/ℏ*Commutator[H, ρ] + Lindbladian[L, ρ];
```

sqlite3 database/catept_complete.db
