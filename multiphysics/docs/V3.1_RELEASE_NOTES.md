## 🚀 QUICK START GUIDE

### Step 1: Download & Verify

```bash
# Download CATEPT-Complete-v3.1.zip (44 MB)

# Verify integrity
md5sum -c CATEPT-Complete-v3.1.zip.md5
sha256sum -c CATEPT-Complete-v3.1.zip.sha256

# Extract
unzip CATEPT-Complete-v3.1.zip
cd CATEPT-Complete-v3.1
```

### Step 2: Choose Your Path

#### Path A: Read the Paper (EASIEST)
```bash
# PDF is already compiled!
open paper/main.pdf  # macOS
xdg-open paper/main.pdf  # Linux
```

#### Path B: Theory Verification
```bash
# Python
cd verification/python
python3 verify_all.py --verbose

# Mathematica
# In Mathematica:
SetDirectory["path/to/CATEPT-Complete-v3.1"];
Get["verification/mathematica/core/CAT_EPT_Unified_v1.0.wl"]

# Lean4
cd verification/lean && lake build
```

#### Path C: Run Simulations ✨ NEW!
```bash
cd simulations/catsim

# Quick validation
make unit_check

# Build database
make db_fast

# Run paper reproduction
make phase3
make phase5
make phase6
make paper_all

# Or all at once
make repro_from_xlsx
```

#### Path D: Explore Examples
```bash
cd simulations/catsim

# TDSE validation
PYTHONPATH=src python examples/quantum_fdtd/run_compare_free.py

# Kwant scattering
PYTHONPATH=src python examples/quantum_fdtd/run_kwant_step.py

# Open quantum (QuTiP)
PYTHONPATH=src python examples/open_quantum/run_qutip_entropic_clock.py

# Entropic time demo
PYTHONPATH=src python examples/entropic/sgi_validation/run.sh
```

## 💻 SYSTEM REQUIREMENTS

### Essential (Minimum)
- **OS:** Linux, macOS, or Windows
- **Python:** 3.8+
- **SQLite:** 3.x

### For Theory
- **LaTeX:** TeX Live 2020+ (for paper)
- **Python:** SymPy, NumPy, pytest
- **Mathematica:** 12.0+ or Wolfram Engine (free)
- **Lean4:** 4.4.0+ (for proofs)

### For Simulations
- **Python:** 3.8+
- **NumPy, SciPy, matplotlib**
- **Optional Backends:**
  - QuTiP (open quantum systems)
  - Kwant (scattering)
  - OQuPy (path integrals)
  - MEEP (electromagnetics)
  - TenPy (tensor networks)
  - Astropy (unit safety)

### Install Simulation Dependencies

```bash
cd simulations/catsim

# Basic requirements
pip install -r requirements-ui.txt

# Optional backends
pip install -e '.[qutip]'   # QuTiP
pip install -e '.[kwant]'   # Kwant
pip install -e '.[oqupy]'   # OQuPy
```
