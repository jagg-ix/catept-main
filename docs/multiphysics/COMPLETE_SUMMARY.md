## 💡 Usage Examples

### **Simple Derivation:**
```mathematica
(* Load framework *)
Get["lib/DerivationFramework.wl"]
Get["lib/LaTeXExporter.wl"]

(* Create derivation *)
d = StartDerivation["Eq22_ComplexAction"]
d = AddStep[d, expr, "Explanation"]
d = FinalizeDerivation[d, result, "QED"]

(* Validate *)
ValidateDerivation[d]

(* Export *)
ExportDerivationToLaTeX[d, "output.tex"]
```
