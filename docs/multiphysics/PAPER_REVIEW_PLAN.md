## Review Methodology

### For Each Turn:

**Checklist Items:**

1. **Equations Review**
   - [ ] All equations have labels (if referenced)
   - [ ] Each equation is introduced in text before appearing
   - [ ] Each equation is explained after appearing
   - [ ] Variables/symbols defined before first use
   - [ ] Physical interpretation provided
   - [ ] Mathematical derivation steps shown or referenced
   - [ ] Units/dimensions consistent
   - [ ] Notation consistent with rest of paper

2. **Figures Review**
   - [ ] Figure has descriptive caption
   - [ ] Figure is referenced in text
   - [ ] Figure is discussed/interpreted
   - [ ] Axes labeled with units
   - [ ] Legend provided if needed
   - [ ] High enough quality/resolution
   - [ ] Colors/symbols explained

3. **Terms Review**
   - [ ] Technical terms defined on first use
   - [ ] Acronyms spelled out on first use
   - [ ] Mathematical notation explained
   - [ ] Physical concepts introduced properly
   - [ ] Jargon minimized or explained

4. **References Review**
   - [ ] Key claims have citations
   - [ ] Novel results clearly marked
   - [ ] Prior work properly credited
   - [ ] Citations formatted consistently
   - [ ] All references in bibliography

5. **Derivations Review**
   - [ ] Assumptions stated clearly
   - [ ] Steps logically connected
   - [ ] Intermediate results shown or referenced
   - [ ] Approximations justified
   - [ ] Limits/domains specified

6. **Physical Interpretation**
   - [ ] Physical meaning of equations explained
   - [ ] Connection to observables discussed
   - [ ] Limits/special cases interpreted
   - [ ] Intuitive understanding provided
   - [ ] Experimental relevance mentioned
